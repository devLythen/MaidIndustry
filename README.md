# Maid Industry

这是一个围绕 **车万女仆** 与 **机械动力** 打造的自动化革命 Minecraft 模组。

## Features

### 祭坛自动化

支持以下两种方式

1. 动力臂祭坛放置
   现在动力臂可以选取祭坛柱作为目标位置，放置物品参与合成
2. 理包机包裹投递
   当鸟居接收到包裹时，会自动拆开包裹，将里面的物品放到祭坛周围的柱子上。

> [!TIP]
> 祭坛自动化会消耗周围八格内任一**神社庭灯**内存储的P点
> 
> 请确保祭坛周围摆放了存有P点的神社庭灯

```bash
./gradlew build
```
